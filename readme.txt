tMagento
========

Talend Magento components

In order to facilitate the communication between the all softwares of organisation information system and Magento we wrappe the Magento API with hree Talend Components :
 * tMagentoSession : allow to open a Magoto API connection (login)
 * tMagentoInput : get entity data from Magento
 * tMagentoOutput : insert / update / delete entity data of Magento
 
This Talend Magento Components embed MagentoJConnector - https://github.com/Altic-labs/MagentoJConnector
